# 🚀 Dijkstra Navigator

An interactive web-based pathfinding application that visualizes the shortest path between two points on a map using graph-based logic.

---

## 🌍 Overview

Dijkstra Navigator allows users to select start and end locations directly on a map and visualize the shortest path between them. The project demonstrates core concepts of graph algorithms combined with real-time UI interaction.

---

## ✨ Features

* 📍 Click-based selection of start and end points
* 🧭 Shortest path visualization on map
* 📏 Distance calculation between points
* ⚡ Real-time backend processing
* 🎨 Clean and responsive user interface

---

## 🛠️ Tech Stack

**Frontend**

* HTML
* CSS
* JavaScript
* Leaflet.js (for map visualization)

**Backend**

* Node.js
* Express.js

**Core Concept**

* Graph Algorithms (Dijkstra - conceptual implementation)

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/tanvichauhan7/Dijkstra-Navigator.git
cd Dijkstra-Navigator
```

---

### 2️⃣ Run the backend server

```
cd backend
npm install
node server.js
```

Server will run at:

```
http://127.0.0.1:5000
```

---

### 3️⃣ Run the frontend

Open the file:

```
frontend/index.html
```

(You can also use Live Server for better experience)

---

## 🧪 How to Use

1. Click anywhere on the map to select the **start point**
2. Click again to select the **end point**
3. Click **Find Path**
4. View the shortest path and distance
